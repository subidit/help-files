## Classification

Today, the most common typeface classifications are the following 5:

1. Serif
2. Sans serif
3. Script
4. Monospaced
5. Display

Commonly, you’ll find groupings such as:

* Serif
* Sans serif
* Slab serif
* Monospaced

Then there are those that are slightly more ambiguous in terms of their potential overlap, but have a shared meaning to most of us, such as:

* Script
* Handwriting
* Display

A more nuanced subgrouping would be like this:

A) **Serif**
 1. Humanist
 2. Garalde
 3. Transitional
 4. Didone
 5. Slab

B) **Sans**
 1. Grotesque
 2. Neo-grotesque
 3. Geometric
 4. Humanist sans

C) Everything else
 1. Script
 2. Graphic
 3. Blackletter
 4. Incised


### Vox-ATypI classification <!-- omit from toc --> 

This is the most official way of classification. It categorises fonts as follows:

1. Classicals
   1. Humanist
   2. Garalde
   3. Transitional

2. Moderns
   1. Didone
   2. Mechanistic
   3. Lineal
      1. Grotesque
      2. Neo-grotesque
      3. Geometric
      4. Humanist

3. Calligraphics
   1. Glyphic
   2. Script
   3. Graphic
   4. Blackletter
   5. Gaelic

4. Non-Latin

1. Size
2. Weight
3. Color
4. Contrast
5. Case
6. Position and alignment
